# 元素水平居中

标签： CSS

---

### 1.行内元素

 - 居中元素：文字、链接等
 - 解决方案：设置上一级父级元素css属性为

> **text-align：center ;**


### 2.块级元素
 - 居中元素：div、p、section等
 - 解决方案：直接设置相应块级元素css属性为

> **margin： 0 auto ;**

 *注意：居中的块状元素需有固定的宽度，否则无法实现居中，因为占据100%的宽度。*

### 3.多个块级元素
 - 居中元素：多个块状元素水平横向排列居中（使用flex布局实现）
 - 解决方案：设置上一级父级元素css属性为

> **display：flex ;**
> **justify-content: center ;**
